Explicação tag < b>

Definição e Uso

A < b > tag especifica o texto em negrito sem nenhuma importância extra.

Nota: De acordo com a especificação do HTML5, a < b > tag deve ser usada como ÚLTIMO recurso quando nenhuma outra tag for mais apropriada. A especificação declara que os cabeçalhos devem ser indicados com as tags < h1 > a < h6 > , o texto enfatizado deve ser indicado com a tag < em > , o texto importante deve ser indicado com a tag < strong > e o texto marcado/destacado deve ser denotado com a tag < mark >.

Dica: Você também pode usar o seguinte CSS para definir texto em negrito: "font-weight: bold;".
